Tecnologias Assistivas - API

Documentação da API Estática para Tecnologias Assistivas

Introdução

A Tecnologias Assistivas - API é uma API estática hospedada em https://sitepublico.github.io/tas-api. Ela fornece acesso a uma coleção de tecnologias assistivas organizadas em arquivos JSON, permitindo consultas por tecnologia individual ou por índices agrupados.

Todos os endpoints estão localizados no diretório /api/.

Endpoints

Tecnologia Individual

URL: /api/{slug}.json

Retorna os detalhes de uma tecnologia assistiva específica, onde {slug} é o identificador único da tecnologia.

Exemplo de Requisição:
GET https://sitepublico.github.io/tas-api/api/leitor-tela.json
Exemplo de Resposta:
{
  "titulo": "Leitor de Tela",
  "descricao": "Software que converte texto em voz para acessibilidade.",
  "orientacao": "Instalar e configurar atalhos.",
  "dicas": "Use fones de ouvido para melhor experiência.",
  "etapas_justificativa": "Acessibilidade para deficientes visuais.",
  "imagem": "../images/leitor-tela.png",
  "categoria": "Acessibilidade Visual",
  "custo": "Gratuito",
  "requer_internet": "Não",
  "plataformas": "Windows, Mac",
  "autor": "Equipe TAS",
  "link": "https://exemplo.com/leitor",
  "etapas": "Ensino Fundamental;Ensino Médio"
}

Índice Geral

URL: /api/indice.json

Lista todas as tecnologias disponíveis em ordem alfabética pelo título.

Exemplo de Requisição:
GET https://sitepublico.github.io/tas-api/api/indice.json
Exemplo de Resposta:
[
  {"titulo": "Ampliador de Tela", "arquivo": "ampliador-tela.json"},
  {"titulo": "Leitor de Tela", "arquivo": "leitor-tela.json"}
]

Categorias

URL: /api/categorias.json

Lista todas as categorias com suas descrições e tecnologias associadas.

Exemplo de Requisição:
GET https://sitepublico.github.io/tas-api/api/categorias.json
Exemplo de Resposta:
{
  "Acessibilidade Visual": {
    "descricao": "Tecnologias para deficientes visuais",
    "tecnologias": [
      {"titulo": "Leitor de Tela", "arquivo": "leitor-tela.json"}
    ]
  },
  "Acessibilidade Auditiva": {
    "descricao": "Tecnologias para deficientes auditivos",
    "tecnologias": [
      {"titulo": "Legendas Automáticas", "arquivo": "legendas-auto.json"}
    ]
  }
}

Custos

URL: /api/custo.json

Lista tecnologias agrupadas por custo.

Exemplo de Requisição:
GET https://sitepublico.github.io/tas-api/api/custo.json
Exemplo de Resposta:
{
  "Gratuito": [
    {"titulo": "Leitor de Tela", "arquivo": "leitor-tela.json"}
  ],
  "Pago": [
    {"titulo": "Software Premium", "arquivo": "software-premium.json"}
  ]
}

Como Usar

Como esta é uma API estática, você pode fazer requisições diretamente usando ferramentas como fetch em JavaScript:

fetch('https://sitepublico.github.io/tas-api/api/leitor-tela.json')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Erro:', error));

Ou em Python com requests:

import requests
response = requests.get('https://sitepublico.github.io/tas-api/api/leitor-tela.json')
data = response.json()
print(data)

Estrutura de Dados

Cada tecnologia individual contém os seguintes campos:

  • titulo: String - Nome da tecnologia
  • descricao: String - Descrição da tecnologia
  • orientacao: String - Instruções de uso
  • dicas: String - Dicas para melhor uso
  • etapas_justificativa: String - Justificativa pedagógica
  • imagem: String - Caminho relativo da imagem (../images/)
  • categoria: String - Categoria da tecnologia
  • custo: String - Custo (ex: "Gratuito", "Pago")
  • requer_internet: String - "Sim" ou "Não"
  • plataformas: String - Plataformas suportadas (separadas por vírgula)
  • autor: String - Autor da tecnologia
  • link: String - URL para mais informações
  • etapas: String - Etapas de ensino (separadas por ponto e vírgula)
